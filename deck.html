01 · Идея

За кем и как мы идём

🎯 Кого копируем
Топ-10 лидеров по «копируемости»: стабильный ROI, loss<13%, направленные (не маркет-мейкеры). Пересматриваем каждый цикл.
⚡ Что копируем
Только СВЕЖИЕ входы лидера — когда он ТОЛЬКО купил и цена ещё у его цены. Эдж не съеден рынком.
🚫 Что НЕ берём
Лайв-матчи (монетки), догон-фавориты, тонкие рынки, крайние цены. Даже трейдер №1 теряет на лайв-матчах.
💰 На чём
Виртуальные $300 (paper-режим). Реальные средства не задействованы — сначала доказываем стратегию.
02 · Инфраструктура

Где всё крутится

🖥 Сервер (VPS, Москва)
CRON каждые 6 минут запускает торговый цикл. xray — прокси для обхода геоблока Polymarket. Всё лёгкое: только cron + прокси.
☁️ Vercel
Витрина — дашборд из 3 страниц. Читает данные напрямую из raw GitHub, обновляется каждые 20 сек без пересборки.
Важно: данные и дизайн разделены — данные живут в git, интерфейс на Vercel. Поэтому правки не мешают торговле, а перегруз сервера исключён.
03 · Движок

Один движок — 6 модулей

Каждые 6 минут cycle.js по очереди запускает:
  • exit-check — вышел ли лидер → выходим следом · тейк-профит 50% на ≥90¢
  • paper-update — марка цен, резолвы, показатели китов, статистика, кривая эквити
  • risk-guard — лимиты день 3% / неделя 8% / просадка 15% → стоп
  • paper-enter — сканит лидерборд, ищет свежие качественные входы по спеку
  • digest — часовая сводка
  • git pull → push — подтягивает правки, публикует данные
04 · Поток данных

Как данные ходят

polycop.fun → лидерборд трейдеров data-api.polymarket → позиции и сделки лидеров ┐ clob.polymarket → стакан (ликвидность) ├→ ДВИЖОК (6 мин) gamma-api → резолвы рынков ┘ │ │ git push ▼ paper-state.json → GitHub (raw) │ дашборд читает каждые 20с ◄┘
paper-state.json — единственный источник истины. Живёт на сервере, публикуется в GitHub, оттуда его напрямую тянет дашборд.

06 · Риск

Три рубежа защиты

🛑 День
−3%
убыток за сутки → стоп на 24 часа
🛑 Неделя
−8%
→ стоп до конца недели + отчёт
🛑 Просадка
−15%
от пика → полная остановка, ручное возобновление
При любой техошибке (не проверить ликвидность, сбой API) — сделка отменяется, а не исполняется вслепую. Правила всегда важнее возможной прибыли.
